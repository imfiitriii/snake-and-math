1.0 Project Description 

  The game is an educational multiplayer turn-based game that aims to enhance user mathematical problem solving ability that can support up to four players which consists of bots and real players. 
This game is called “Snake and Math” which is inspired by the classic Snake and Ladder board game. As players progress onto higher board positions, the difficulty of the questions increases  to make it more challenging over time and enhance user solving skill gradually.

At the start of the game, users must select the number of players up to four players with the minimum requirement of one player. If the user selects less than four players, then the remaining space will be automatically occupied by the bots. 

Players can also choose the difficulty level of the bots, ranging from easy, medium to hard. These difficulty levels determine the probability of the bots answering questions correctly. For instance, bots with easy difficulty will have a higher probability of answering the questions incorrectly.

The game begins with the first player taking a turn. Before rolling the dice, players are required to answer the mathematical question. After inserting the answers, the dice will roll automatically. 
If the answer is wrong the player will be given a penalty by moving backwards based on the number on the dice, while the correct answer allows the user to progress forward. These rules enhance the experience, compared to classical snake and ladder games.

Additionally, each player is given a time limit to answer the question. If the player fails to answer before the times end, the penalty is the same as an incorrect answer applied, which is the dice will be rolled automatically and the user will move backwards.

Moreover, This game also added some twist where there willbe power ups that can be used by player to assist them in the game. The power ups are Time Freeze and Snake Blocker where to obtain Time Freeze, player must answer 5 consequtive question correctly while Snake Blocker will be generated randomly on the tiles and landing on them allow it to be used by the player.

Lastly, the game also includes the classical uniqueness of Snake and Ladder mechanics. Landings on ladders allow players to move forward to a higher position while landing on a snake causes the player to move backwards.
